1. Была создана сеть
    docker network create app-net

2. Запуск контейнера с БД
    docker run -d -p 27017:27017 --network=app-net --name my-mongo mongo:3

3. Создание образа своего контейнеа и его Запуск
    sudo docker build -t my-image .
    sudo docker run -d -p 3003:3003 --name my-container --network=app-net my-image
